# Sailaway - GIT Anleitung

    1) download GIT from git-scm.com?

    2) register on github.com?

    3) create a new folder somewhere on your computer?
       (e.g. GIT)?

    4) right click on the new folder?

    5) select the option “GIT Bash Here”?

    6) clone remote repository (git clone <Link>)?

    7) Select the cloned folder “Sailaway”(cd Sailaway)?

    8) check the status (git status)?

    9) directly create/save new files in the folder “Sailaway”?

    10) add the new file (git add <filename>)?

    11) add a commit (git commit –m “Text”)?

    12) push your local repository into the remote one (git push)?

    13) pull, to update you local (cloned) repository (git pull)
